# alarm clock for YSWS on hackclub
Its a alarm clock, that has simple design its powered by usb-c it has extremaly loud sound mady by personal alarms(suposedly 140DB). To turn it off you either have to answer for preaty easy question or play one of minigames(crossy road inspired one, draw 94% accurate: circle, rectangle, star or cut the object almost perfectly in half) the questions or game are randomally selected from a list every time allarm runs. The clock uses its own wi-fi captive portal to run the site.
